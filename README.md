# Secret santa
## Principe du concept
Concept (pas) original de distribution de cadeaux entre amis. Chaque personne renseignée doit faire un cadeau à une autre personne aléatoirement. Ne va que dans un sens (l'autre personne donnera son cadeau à une autre personne).
